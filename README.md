# Чтение переменных среды

Переменные среды читаются либо из окружения, либо из config.cfg
Приоритет чтения:

1. Переменные окружения;
2. Конфигурационный файл `config.cfg`.

Переменные окружения генерируются по формуле: `TENTACULA__<РАЗДЕЛ>__<ПЕРЕМЕННАЯ>`

# Установка
Комплекция пакета Conda. Последовательность:
1. Исполнить `bash ./build.sh`;
2. Возможно потребуется инициализировать shell (bash). Необходимо последовательно исполнив команды `conda init bash` и `exec bash`.
